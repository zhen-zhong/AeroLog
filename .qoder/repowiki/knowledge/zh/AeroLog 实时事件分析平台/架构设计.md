- **统一基础设施底座**：通过 `deploy/docker-compose.yml` 集中编排 PostgreSQL（元数据）、ClickHouse（事件存储）、Redpanda（消息总线）及可观测性组件，为所有子模块提供标准化的运行环境。
- **跨语言工作区协同**：采用 Go Workspace (`server/go.work`) 统一管理 API、Collector、Consumer 及共享包 (`pkg`)，确保后端微服务间数据模型与通信契约的强一致性；同时通过 NPM 管理 Web 前端与 Web SDK 的依赖。
- **全局任务调度**：利用根目录 `Makefile` 封装 `dev`、`infra`、`test` 等跨模块命令，屏蔽底层各技术栈（Go/Node/Docker）的启动差异，实现一键式全栈开发与环境初始化。
