- 入口层：`cmd/main.go` 负责初始化 PostgreSQL/ClickHouse 连接池，配置 CORS 与指标中间件，并注册路由。
- 配置层：`internal/config` 通过环境变量注入运行时配置（数据库地址、JWT 密钥等）。
- 业务层：`internal/handler` 采用 Handler 结构体模式，直接嵌入数据库连接依赖，并在 `Register` 方法中定义 REST 路由映射。
- 数据访问：直接在 Handler 方法内编写 SQL/ClickHouse 查询逻辑，未抽象独立的 Repository 层。
- 依赖方向：入口 -> 配置/处理器 -> 共享包 (`pkg/metrics`) 及第三方驱动。
