- **分层结构**：模块内部按功能域划分为 `model`（数据契约）、`mq`（消息中间件适配）和 `metrics`（可观测性基础设施）三个子包。
- **数据契约层**：`model` 包定义了 SDK 上报的原始事件 `Event` 及经 Collector 包装后的 `EnvelopedEvent`，并提供基础的字段校验逻辑，作为跨服务的数据交换标准。
- **基础设施适配层**：`mq` 包基于 `sarama` 库封装了异步 Kafka 生产者，内置 Snappy 压缩、批量刷新及错误通道消费逻辑；`metrics` 包基于 `prometheus/client_golang` 提供全局注册表及便捷的指标创建函数，并通过独立的 HTTP Server 暴露 `/metrics` 端点以实现监控隔离。
