- **分层架构**：模块内部划分为 `components`（UI 表现）、`features`（业务页面组合）和 `lib`（工具与数据访问）三层。
- **布局体系**：以 `AppShell.tsx` 为根容器，嵌套 `DashboardShell` 实现响应式侧边栏导航与顶部 Header，通过 `useUIStore` 管理折叠状态。
- **组件设计**：`ui/` 目录存放基于 `class-variance-authority` (CVA) 的原子组件（如 Button, Card），支持变体配置；`layout/` 和 `react-bits/` 提供高阶布局与动效组件。
- **数据交互**：`lib/api.ts` 封装统一的 `req` 请求函数，集中定义 TypeScript 接口并暴露 RESTful 方法，供 `features` 中的 React Query 钩子调用。
