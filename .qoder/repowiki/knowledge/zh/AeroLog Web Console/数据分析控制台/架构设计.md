- **路由层**：采用 Next.js App Router，每个子目录（如 `event/`, `funnel/`）对应一个独立的路由页面 (`page.tsx`)。
- **UI 组件层**：高度复用 `@/features/analytics/analytics-ui` 中的原子化组件（如 `ChartPanel`, `MetricTile`, `ReportControls`），确保界面风格与交互逻辑的一致性。
- **数据交互层**：统一通过 `@/lib/api` 封装的客户端 API 进行数据请求，结合 `@tanstack/react-query` 管理异步状态、缓存及自动刷新（如实时页面的 15s 轮询）。
- **可视化层**：使用 `echarts-for-react` 实现动态图表渲染，并通过 `next/dynamic` 进行服务端渲染（SSR）隔离以优化性能。
