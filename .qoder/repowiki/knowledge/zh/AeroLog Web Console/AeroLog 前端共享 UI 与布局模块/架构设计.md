- **入口与布局**：`AppShell.tsx` 作为根级包装器，组合 `Providers` 与 `DashboardShell`；`dashboard-shell.tsx` 实现响应式侧边栏（桌面端固定/移动端 Sheet）及基于 `next/navigation` 的路由高亮逻辑。
- **组件分层**：`components/ui/` 存放基于 `class-variance-authority` (CVA) 和 `@radix-ui/react-slot` 的原子 UI 组件（如 Button, Card, Dialog）；`components/layout/` 提供页面级结构件（PageHeader）；`components/react-bits/` 包含动效增强组件。
- **业务特征 UI**：`features/analytics/analytics-ui.tsx` 封装了分析场景专用的复合组件（如 DateTimeRange, EventRankList, ChartPanel），实现了筛选器工具栏与指标卡片的标准化视觉模式。
- **基础设施**：`lib/api.ts` 定义了统一的 RESTful 客户端及 TypeScript 接口契约；`lib/utils.ts` 提供 `cn` 工具函数以支持 Tailwind CSS 类名的动态合并。
