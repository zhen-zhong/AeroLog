- 所有交互型 UI 组件均使用 "use client" 指令标记，并采用 React.forwardRef 配合 displayName 以确保可访问性与调试友好性。
- 样式处理统一通过 lib/utils 中的 cn 函数结合 clsx 与 tailwind-merge，支持动态变体与外部类名覆盖。
- 复杂表单与数据展示页面遵循“桌面端表格 + 移动端卡片”的双轨渲染模式，通过 hidden md:block 等断点类实现响应式切换。
- API 请求统一由 lib/api.ts 中的 req 函数拦截，自动添加 /v1 前缀、JSON 头及 no-store 缓存策略，并集中处理非 200 状态码错误。
