- **统一布局架构**：在 `src/app/layout.tsx` 中定义根布局，强制所有子页面（包括 `/console` 和 `/admin`）包裹于 `AppShell` 组件内。
- **全局状态与 UI 框架**：`AppShell` 集中配置 Ant Design 主题及 TanStack Query 客户端，确保跨模块的数据请求缓存与 UI 风格一致性。
- **集中式路由导航**：侧边栏菜单在 `AppShell` 中统一定义，将数据分析（Console）与系统管理（Admin）路径进行物理隔离但逻辑聚合。
- **标准化 API 交互**：通过 `src/lib/api.ts` 提供统一的 RESTful 客户端，封装了针对后端 Go 服务的认证、错误处理及毫秒级时间戳转换逻辑。
