- 所有 UI 原子组件均使用 `React.forwardRef` 进行封装，并显式设置 `displayName` 以支持调试与 DevTools 识别。
- 业务页面采用“桌面端表格 + 移动端卡片”的双轨响应式布局模式，通过 `hidden md:block` 等 Tailwind 类名控制不同视口下的组件显隐。
- API 客户端统一返回包装后的泛型结构（如 `ApiList<T>` 或 `ApiOne<T>`），并在业务层通过 `data?.data` 提取实际业务数组或对象。
- 复杂交互场景（如新建项目）优先使用 `Dialog` 模态框承载表单，并结合 `react-hook-form` 或原生 `useState` 管理表单状态与错误提示。
