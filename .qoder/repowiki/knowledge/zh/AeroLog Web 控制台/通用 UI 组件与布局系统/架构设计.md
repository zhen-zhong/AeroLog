- **分层结构**：模块内部划分为 `components/ui`（原子 UI 组件）、`components/layout`（页面布局壳）、`features`（业务页面入口）和 `lib`（工具与 API 客户端）。
- **布局边界**：`AppShell.tsx` 作为根容器，组合 `Providers` 与 `DashboardShell`；`DashboardShell` 实现了响应式侧边栏导航（桌面端固定、移动端 Sheet 抽屉），并通过 `usePathname` 实现路由高亮。
- **数据流设计**：`lib/api.ts` 封装了基于 `fetch` 的统一请求客户端，定义了项目、事件、属性等核心业务的 TypeScript 接口，并暴露 `api` 对象供 `features` 下的页面通过 `@tanstack/react-query` 调用。
- **样式体系**：采用 Tailwind CSS 配合 `class-variance-authority` (CVA) 管理组件变体，通过 `lib/utils.ts` 中的 `cn` 函数实现类名合并与冲突处理。
