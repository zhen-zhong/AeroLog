- **分层结构**：模块内部划分为 `components/ui`（原子 UI 组件）、`components/layout`（页面级布局壳）、`features`（业务视图组合）和 `lib`（工具与 API 层）。
- **布局架构**：采用 `AppShell` 作为根容器，嵌套 `DashboardShell` 实现响应式侧边栏导航（桌面端固定、移动端 Sheet 抽屉），通过 `usePathname` 实现路由高亮。
- **数据交互**：`lib/api.ts` 封装了基于 `fetch` 的统一请求客户端，定义了项目、事件、用户等核心业务的 TypeScript 接口，并与 `@tanstack/react-query` 在业务层配合使用。
- **样式体系**：基于 Tailwind CSS，通过 `lib/utils.ts` 中的 `cn` 函数合并 `clsx` 与 `tailwind-merge`，确保组件样式的可覆盖性与一致性。
