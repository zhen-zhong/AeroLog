- **全局状态与依赖注入**：在 `AppShell` 中统一包裹 `Providers`（React Query）与 `DashboardShell`，确保所有子页面共享相同的数据获取缓存策略与 UI 框架。
- **标准化 API 契约**：通过 `src/lib/api.ts` 定义统一的 RESTful 客户端，强制所有业务模块（admin/console）使用一致的毫秒级时间戳与 `/v1/projects/{id}` 路径规范与后端交互。
