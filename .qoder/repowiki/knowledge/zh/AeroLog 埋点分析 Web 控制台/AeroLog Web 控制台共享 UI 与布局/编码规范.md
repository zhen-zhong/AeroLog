- UI 组件使用 `cva` 定义变体（variants）和尺寸（size），并通过 `cn` 工具函数合并默认类名与用户传入的 `className`。
- API 请求统一通过 `lib/api.ts` 中的 `req` 辅助函数发起，自动添加 `/v1` 路径前缀并设置 `Content-Type: application/json`。
- 页面组件通常以 `PageHeader` 开头，后接指标卡片（Metric Cards）和主要内容区域（表格或卡片列表），并区分桌面端与移动端视图。
- 所有交互性组件（如导航、页面逻辑）均标记为 `"use client"` 以适配 Next.js 服务端渲染架构。
