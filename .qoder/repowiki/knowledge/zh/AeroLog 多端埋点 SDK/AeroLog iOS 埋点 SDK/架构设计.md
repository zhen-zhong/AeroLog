- **模块化封装**：基于 Swift Package Manager (`Package.swift`) 构建，核心逻辑位于 `Sources/AeroLog`。
- **核心组件**：`AeroLog` 作为单例门面协调工作流；`AeroConfig` 定义配置契约；`EventStore` 实现基于 NDJSON 文件的轻量级离线存储。
- **数据流设计**：采用内存缓冲（`buffer`）与磁盘持久化（`EventStore`）双层机制，通过 `NSLock` 保证并发安全，利用 `Timer` 和生命周期监听触发异步刷新。
- **依赖管理**：仅依赖 Foundation 和 UIKit（条件编译），无第三方库依赖，确保集成轻量化。
