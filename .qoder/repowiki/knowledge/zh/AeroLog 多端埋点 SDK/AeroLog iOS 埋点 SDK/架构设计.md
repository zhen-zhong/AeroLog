- **核心架构**：采用单例模式 (`AeroLog.shared`) 作为统一入口，内部通过 `NSLock` 保证多线程安全。
- **数据流**：事件经 `enqueue` 组装（注入 Super Properties、系统信息、Session ID）后进入内存缓冲；满足阈值或定时触发 `flush`。
- **持久化层**：`EventStore` 基于 Application Support 目录下的 NDJSON 文件实现简易队列，支持容量限制与失败重试。
- **生命周期管理**：利用 `UIApplication` 通知监听应用前后台切换，实现自动采集启停事件及后台主动刷盘。
- **网络通信**：使用 `URLSession` 进行 JSON 批量 POST 上报，并通过信号量同步等待响应以决定本地数据的保留或删除。
