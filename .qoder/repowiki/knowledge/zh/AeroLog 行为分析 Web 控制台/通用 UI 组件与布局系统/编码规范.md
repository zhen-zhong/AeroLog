- 原子组件通过 `cva` 定义样式变体，并使用 `React.forwardRef` 配合 `displayName` 进行封装，支持 `asChild` 属性以增强组合能力。
- 业务页面统一采用 `PageHeader` 定义标题与操作区，并结合 `Card`、`Table` 等组件构建内容，同时通过 `Skeleton` 实现加载态占位。
- API 客户端在 `lib/api.ts` 中集中定义，所有接口返回遵循 `ApiList<T>` 或 `ApiOne<T>` 的统一包装结构，路径拼接逻辑内置于 `req` 函数。
- 样式类名合并统一使用 `cn(...)` 工具函数，确保 Tailwind 类冲突时能按优先级正确合并，并支持条件渲染类名。
