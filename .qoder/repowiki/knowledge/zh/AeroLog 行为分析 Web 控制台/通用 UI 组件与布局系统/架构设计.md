- **分层结构**：模块内部划分为 `components/ui`（原子 UI 组件）、`components/layout`（页面布局与导航）、`features`（业务视图组合）和 `lib`（工具与 API 客户端）。
- **布局架构**：采用 `AppShell` 作为根容器，嵌套 `DashboardShell` 实现响应式侧边栏（支持折叠与移动端 Sheet 菜单）和顶部导航，通过 `useUIStore` 管理全局 UI 状态。
- **数据流**：`lib/api.ts` 封装了基于 `fetch` 的统一请求客户端，定义了大量 TypeScript 接口；业务页面（如 `projects-page.tsx`）通过 `@tanstack/react-query` 消费这些 API。
- **样式体系**：所有组件均基于 Tailwind CSS，并通过 `lib/utils.ts` 中的 `cn` 函数合并 `clsx` 与 `tailwind-merge`，确保样式覆盖的正确性。
