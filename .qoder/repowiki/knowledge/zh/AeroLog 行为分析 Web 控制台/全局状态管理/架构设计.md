- 采用扁平化架构，每个 Store 独立定义（`project-store.ts`, `ui-store.ts`），通过 `zustand` 的 `create` API 暴露 React Hooks。
- 集成 `zustand/middleware` 中的 `persist` 中间件实现状态持久化，并通过自定义 `noopStorage` 适配 SSR 环境，防止服务端访问 `localStorage` 报错。
- `ui-store.ts` 内部封装了 DOM 操作逻辑（`applyThemeClass`），在状态变更时同步更新 `document.documentElement` 的类名与样式，实现主题切换的副作用管理。
