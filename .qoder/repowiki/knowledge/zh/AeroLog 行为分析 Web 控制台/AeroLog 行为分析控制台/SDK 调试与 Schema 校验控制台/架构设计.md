- **入口与状态管理**：以 `page.tsx` 为单一入口，采用 React Hooks (`useState`, `useQuery`, `useMutation`) 管理本地 UI 状态与远程数据同步。
- **数据交互层**：通过 `@/lib/api` 封装的客户端与后端通信，利用 `@tanstack/react-query` 处理异步请求、缓存及失效刷新（如保存规则后自动刷新调试列表）。
- **UI 架构**：基于 shadcn/ui 组件库构建响应式布局，左侧为过滤条件与 Schema 规则配置面板，右侧为统计概览与双标签页（事件流/问题表）展示区。
- **业务逻辑**：内置参数摘要生成、时间线跳转链接构造及校验结果徽章渲染等辅助逻辑，实现从原始数据到可视化诊断信息的转换。
