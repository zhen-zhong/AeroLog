- **路由编排**：采用 Next.js App Router (`web/src/app/console/`) 实现模块化页面路由，各子路径（如 `/debugger`, `/funnel`）对应独立业务场景。
- **共享基础设施**：所有子模块统一依赖 `@/lib/api` 进行后端通信，并复用 `@/features/analytics/analytics-ui` 中的标准化 UI 组件（如 `AnalyticsHeader`, `ChartPanel`, `MetricTile`），确保视觉与交互一致性。
- **状态管理**：通过 `@/stores/project-store` 实现跨页面的项目上下文同步，确保用户在切换分析维度时保持当前选中项目的一致性。
- **数据可视化**：集成 `echarts-for-react` 并通过 `next/dynamic` 进行客户端动态加载，优化首屏性能并支持复杂的趋势图、漏斗图及热力图渲染。
